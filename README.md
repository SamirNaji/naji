# Profile Samir Naji

A professional bilingual portfolio website for Samir Naji (Ghazal), a creative Photographer and Designer.

## Features
- **Bilingual Support**: English and Arabic with RTL support.
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
- **Modern UI**: Built with Tailwind CSS and custom animations.
- **Easy Customization**: All text is managed in a single JavaScript object.

## How to Deploy to GitHub Pages

1.  Create a new repository on GitHub (e.g., `my-portfolio`).
2.  Open the folder containing these files in your terminal.
3.  Run the following commands:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/my-portfolio.git
git push -u origin main
```

4.  Go to your GitHub Repository Settings -> Pages.
5.  Select `main` branch and save.
6.  Your site will be live at `https://YOUR_USERNAME.github.io/my-portfolio/`.
